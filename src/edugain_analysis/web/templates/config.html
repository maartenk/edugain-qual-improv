⚙️ Analysis Settings
ℹ️ About Settings

These settings control the behavior of the analysis and caching system:

  • Auto-Refresh Interval: Reserved for future auto-refresh feature
  • Metadata Cache Expiry: Controls how long eduGAIN metadata is cached
  • URL Validation Timeout: Maximum time to wait for URL validation responses
  • URL Validation Threads: Number of parallel threads for URL validation (higher = faster but more load)
  • Federation Cache Expiry: How long federation name mappings are cached

Note: Settings changes only affect new data imports. Run python -m edugain_analysis.web.import_data to apply changes.
